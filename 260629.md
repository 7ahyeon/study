*chmod*
- 권한 부여
- 읽기(r) 쓰기(w) 실행(x)

*&*
- 백그라운드 실행

*pwd*
- 현재 위치
- print working directory

*mkdir*
- 디렉터리 생성

*cd*
- cd ~ 홈 디렉터리 이동
- cd .. 상위 디렉터리 이동

*ls*
- ls -alh 숨김 파일 및 파일 사이즈 조회

*tree*
- 디렉터리/파일 구조 확인
- tree -d 디렉터리만
- tree -a 숨김파일까지
- tree -L {깊이} {디렉터리} 깊이 지정 조회

*cp*
- 디렉터리/파일 복사
- cp {원본파일}{위치}
- cp {원본파일}{위치/다른이름파일명}
- cp -r {원본디렉터리위치}{디렉터리위치} 디렉터리 복사
- cp -r {원본위치}. 현재디렉터리로복사

*mv*
- 디렉터리/파일 이동

*rm*
- 디렉터리/파일 삭제
- rm -r 디렉터리 삭제
- rm -rf 파일 강제 삭제

*vi*
- i : INSERT 모드 : 문자 편집
- esc : COMMAND 모드 : 복사/붙여넣기/파일저장/종료 등
- :q + enter : 종료
- wq + enter : 저장 종료
- wq! + enter : 저장 강제 종료
- dd : 줄 삭제
- yy : 복사
- p : 붙여넣기
- u : 되돌리기
- :set number : 라인 출력
- /검색어 n : 텍스트 찾기
- :%s/원본/대체/g : 텍스트 대체

- cat : 파일 내용 확인/출
- grep : 출력 내용 검색
- cat 파일명 | grep 검색어 : 파일 내용 검색
- grep -B n 검색어 : 파일 내용 검색 + 이전 n개 라인 같이 출력
- grep -A n 검색어 : 파일 내용 검색 + 이후 n개 라인 같이 출력
- grep -i 검색어 : 대소문자 없이 검색
- 이전 명령어 > 파일명 : 이전 명령어 출력을 파일로 저장(기존 파일 있을 경우 덮어쓰기)
- 이전 명령어 >> 파일명 : 이전 명령어 출력을 파일로 저장(기존 파일 있을 경우 내용 추가)
- less : 용량 큰 파일 읽기(읽기 전용/편집x)

- tar : 아카이브 (여러 디렉터리/파일 하나의 파일로 묶음/압축X)
- tar -cvf {파일명.tar}{디렉터리/파일} : 파일/디렉터리 아카이브 생성
- tar -xvf {파일명.tar} : 파일/디렉터리 해제
- tar.gz : 아카이브 압축
- tar -czvf {파일명.tar.gz}{디렉터리/파일} : 파일/디렉터리 아카이브 압축 생성
- tar -xzvf {파일명.tar.gz}{디렉터리/파일} : 파일/디렉터리 아카이브 압축 해제

*명령어*
- ipconfig : ip 확인
- chmod + x : 프로세스 실행
- ps -al :  프로세스 상태 확인
- export 변수명=변수값 : 환경변수 지정
- watch -n 주기 명령어 : 주기적 명령어 실행
- alias 단축어=명령어 : 명령어 단축키 지정

*명령어 환경 설정*
- 사용자 홈 디렉터리 .zshrc/.bashrc 파일 수정
- vi ~/.zshrc
- vi ~/.bashrc
